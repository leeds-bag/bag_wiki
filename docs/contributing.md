# Contributing

Thank you for contributing! Use this short checklist:

- Create or update a Markdown file in docs/
- Use headings and short sections for readability
- Add images to docs/images/ and reference with relative paths
- Submit a branch and open a PR against main

Suggested PR title format: "docs: add/modify <topic>"

Template for a new how-to:
```md
# Short title

## Why
Brief reason / context.

## Steps
1. Step one
2. Step two

## Notes
- Extra tips or links
```
